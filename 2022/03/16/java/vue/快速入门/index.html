快速入门

  • Vue

    一套前端框架，免除原生JS中的DOM操作，简化书写。
    其中，DOM为文本对象模型，可以访问和改变HTML中所有元素。

  • MVC：只能实现模型到视图的单项展示
  • MVVM：可以实现数据的双向绑定，数据模块(model)和页面(View)通过ViewModel实现双向绑定，模型和视图就可以同时变化。
